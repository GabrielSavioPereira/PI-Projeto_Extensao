# NexStock

Sistema web/mobile de controle de estoque e gestão financeira desenvolvido como Projeto Integrador (PI) para a faculdade.

## 📌 Sobre o projeto

O *NexStock* surgiu da necessidade de digitalizar e otimizar o controle de estoque e financeiro de uma loja de roupas real, atualmente administrada de forma manual.

O sistema está sendo desenvolvido para auxiliar no gerenciamento de:

- Cadastro de produtos
- Controle de entrada e saída de estoque
- Gestão financeira
- Relatórios de vendas
- Controle de clientes e fornecedores

Além do contexto acadêmico, o projeto possui aplicação prática, pois foi idealizado com base nas necessidades reais de uma loja de vestuário.

## 🎯 Objetivo

Desenvolver uma solução prática, intuitiva e acessível para melhorar a organização operacional e financeira de pequenas lojas de roupas.
